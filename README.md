# Single-wood-matching
2026.1.14 上传单木匹配2.0

添加了两个功能

1.交互式干扰剔除启动之前，删去离最近实测点超过xm的单木分割点x可以自由设置（默认为3）

2.进行交互式干扰剔除时，当鼠标悬浮在点上时，显示树高


2026.1.15 上传单木匹配2.1

添加三个功能，删去一个功能

添加 在交互界面中，鼠标右键选择某点后，显示与这个点可能匹配的所有点的连线（距离小于3，树高差小于5）

添加 根据树高、胸径分级的分级精度分析

添加 交互式删除的“撤销/恢复”功能。再次圈选中那些已经变红（标记为删除）的点，它们会恢复为蓝色（取消删除）。

删除 点云分割参数调整建议中的树高问题检测


2026.1.15 上传单木匹配3.0

使用匈牙利算法代替贪心算法进行匹配（计算复杂度高、高密度林分慎用！）


2026.1.22

新增功能：
clean_isolated_points 函数现在同时检查距离和树高。
删除规则：如果一个分割点，在设定的距离范围内（如3米），找不到任何一个高度差异在允许范围内（如5米）的实测点，它就会被判定为干扰点并删除。
